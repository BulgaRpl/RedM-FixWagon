# RedM-FixWagon

A little script to automatically remove buggy wagon (when horse walk in place).  
Resmon : 0.00ms

# How to install :
1. Drag & drop the "kd_fixwagon" folder in your resources
3. Add "ensure kd_fixwagon" in your server.cfg   

Done !
